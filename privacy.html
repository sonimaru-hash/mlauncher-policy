プライバシーポリシー

1. はじめに

本プライバシーポリシーは、MLauncher（以下、「本アプリ」）が提供するサービスにおける、ユーザーの情報の取り扱いについて説明するものです。

2. 取得する情報および利用目的

本アプリは、GoogleカレンダーAPIを利用して以下の情報を取得しますが、これらは記載された目的の範囲内でのみ利用されます。

  • カレンダーの予定情報（タイトル、日時、場所など）:
    利用目的：本アプリ内でのスケジュール表示およびカスタム通知機能の提供のため
  • カレンダーのID:
    利用目的：特定のカレンダーを識別し、同期を維持するため

3. データの保存と保護

本アプリは、Googleから取得したカレンダーデータを、ローカル環境（ユーザーのPC内）でのみ一時的に保持し、外部サーバーへ保存・送信することはありません。

ユーザーが本アプリとの連携を解除した場合、取得したデータは速やかに破棄されます。

4. データの第三者提供

本アプリは、取得したユーザーデータを第三者に販売、共有、または提供することはありません。ただし、法令に基づく場合を除きます。

5. Google APIに関する声明（重要）

本アプリによる Google API から受け取った情報の使用および他のアプリへの転送は、Google API サービス ユーザーデータに関するポリシー（限定公開の要件を含む）に従います。
